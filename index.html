Dependencies

  • BASS and pybass (HELL YEAH)
  • pyqt4 ("nuke on ants" or "the frontpage of python gui dev")
  • mutagen (cool name and can also read song tags)
  • pyhook (A link to the past)
  • python-lastfm
  • bottlenose

Build

You will need pyinstaller on the same folder as the source code, already configured. Just run build_win32.bat and it will do all the work for you. (It won't work on Linux) (It won't work on Macs) (It may not work with ActiveState Python)
